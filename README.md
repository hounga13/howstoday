# 🔮 오늘 어때? (How's Today?) 

서버 없이 가볍게 운영하며 광고 수익을 목적으로 하는, "여기 어때?" 패러디 운세 웹페이지를 위한 기획서입니다.

---

## 📋 요구사항 명세서 (Requirements Specification)

### 1. 목표 및 타겟
- **목표**: 별도의 유지보수와 서버 비용 없이 완전한 정적 웹사이트로 동작하며, 꾸준한 트래픽을 유도해 광고 수익 창출.
- **타겟**: 매일 아침 가볍게 운세를 확인하고자 하는 10~30대 유저. 가볍고 재미있는 컨셉 선호.

### 2. 핵심 기능 (Functional Requirements)
1. **사용자 정보 입력**:
   - 사용자의 이름, 생년월일(또는 띠/별자리)을 입력받는 폼 제공.
   - 복잡한 회원가입이나 로그인 절차 일체 없음.
2. **오늘의 운세 생성 (무서버 구현)**:
   - 서버 DB 조회나 외부 유료 API를 사용하지 않습니다.
   - **구현 방식**: `현재 날짜` + `사용자의 정보(이름, 생일)`를 조합하여 시드(Seed) 값으로 쓴 후, 미리 준비된 운세 문구 풀(Pool)에서 결정론적 난수를 통해 결과를 매칭합니다. (이렇게 하면 같은 날, 같은 사용자는 항상 같은 결과를 보게 되어 신뢰성을 줍니다).
   - 운세 종류: 총운, 금전운, 연애운 등.
3. **결과 공유 (바이럴)**:
   - "친구에게 공유하고 야속한 내 운세 탓하기" 등의 재미있는 문구와 함께 카카오톡 공유 기능 및 링크 복사 기능 제공.
   - 이를 통해 추가 트래픽 직접 유도.
4. **광고 지면 (Monetization)**:
   - 운세 결과를 보기 직전, 혹은 운세 결과 하단에 Google AdSense 등 배너를 노출.

### 3. 비기능 요소 및 배포환경 (Non-functional Requirements)
- **배포 방식**: GitHub Pages, Vercel 등을 통한 100% 무료 정적 배포 (HTML, CSS, JS만 사용).
- **반응형 웹**: 철저한 **모바일 퍼스트(Mobile First)** 설계.

## 🎨 디자인 명세서 (Design Specification)

### 1. 전반적인 톤 앤 매너 (Tone & Manner)
- **모티브**: 여행/숙박 플랫폼 "여기 어때?"의 메인 화면.
- **분위기**: 친절하고, 에너지 넘치며, 누구나 쉽게 누를 수 있을 만큼 직관적인 UI.

### 2. 컬러 팔레트 (Color Palette)
- **Primary Color**: 🔴 밝고 쨍한 레드. 브랜드의 핵심 컬러 (#F7323F 또는 유사한 진홍색)
- **Background Color**: ⚪ 깔끔한 화이트 (#FFFFFF) 및 콘텐츠 구분을 위한 연한 회색 (#F2F4F5)
- **Text Color**: 짙은 그레이 (#333333)로 가독성을 높임.

### 3. 타이포그래피 (Typography)
- **로고 및 제목 폰트**: 둥글고 장난스러운 동시에 볼드한 폰트 (예: **'여기어때 잘난체'** 무료 폰트 활용)
- **본문 폰트**: 정보 전달을 위해 깔끔한 구글 웹폰트 **'Pretendard(프리텐다드)'** 활용.

### 4. UI 및 레이아웃 구조 (Mobile 화면 기준)

**① 상단 영역 (Header & Hero)**
   - 화면 중앙 상단에 "오늘 어때?" 로고 배치 (빨간 색상, 잘난체 폰트).
   - 바로 아래 메인 카피: *"오늘 하루, 내 운세 어때?"*
   - 귀여운 이모지(🔮, 🍀 등)나 일러스트를 작게 배치.

**② 입력 영역 (Input Card - 여기어때 호텔 검색폼 패러디)**
   - 흰색 둥근 카드(Border-radius: 12px) 안에 그림자가 옅게 깔린 박스 디자인.
   - **이름 입력 박스**: 터치하기 넉넉하게 큰 텍스트 폼.
   - **생일/별자리 선택**: "여기 어때"의 날짜 선택 창처럼 생겨 호기심을 유발.
   - **운세보기 버튼**: 카드 최하단에 폭 100%로 가득 차는 **빨간색의 둥근 대형 버튼** 제공. (텍스트: "오늘의 운세 물어보기")

**③ 광고 영역 (Ad Area)**
   - 정보 입력 폼 바로 아래 "추천 혹은 이벤트" 배너처럼 보이게 광고 위치 (Google AdSense 등을 삽입할 수 있는 빈 영역).

**④ 운세 결과 영역 (Result Section)**
   - 팝업 형태로 뜨거나 다음 페이지 느낌으로 전환(SPA 방식).
   - 상단에 크게 점수 표시: "오늘의 총운: 95점! 훌륭해✨"
   - 재미있는 운세 요약 글을 카드 형태로 3~4개 짧게 나열.
   - 하단 버튼 2개 나란히 배치:
      - `[빨간버튼]` 카카오톡으로 결과 공유하기
      - `[회색버튼]` 다시하기
